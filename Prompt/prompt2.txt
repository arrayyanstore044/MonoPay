Isi dari planningnya adalah sebagaia berikut :

Buatkan API untuk registrasi user baru yang mana nanti akan ada email verification untuk verifikasi user tersebut

Endpoint : POST /api/auth/register

Request Body :
 {
    "name" : "Jhon",
    "notelp" : "08123456789",
    "email" : "[EMAIL_ADDRESS]",
    "password" : "password"
 }

Response Body (Success) :
 {
    "status" : "OK!"
    "message": "User successfully registered!"
    "data": {
        "user": {
            "id": 1,
            "name": "Jhon",
            "notelp" : "08123456789",
            "email": "[EMAIL_ADDRESS]",
            "password": "password"
        }
    }
 }

 Response Body (Email Already Exist) :
 { 
    "status": "ERROR",
    "message": "Email already exists.",
    "data": null
}
